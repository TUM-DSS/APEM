# Introduction

APEM is a framework for allocation and pricing in electricity markets.

It supports two market modeling workflows:

- `unit_based_model`
- `order_book_based_model`

At a high level, APEM provides:

- Scenario parsing and data preparation.
- Allocation and pricing optimization pipelines.
- Post-run analysis utilities.
- Scriptable entrypoints for experiments and comparisons.

For first execution steps, continue with the installation page and then the existing getting started guide.
